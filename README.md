# -znsb-
这是一个DIY智能手表
主控芯片：基于STM32F103系列的芯片，
开发框架：基于 STM32 标准外设库，模块化驱动设计
显示设备：OLED 显示屏（I2C/SPI 接口，128×64 分辨率）
交互方式：4 个独立按键（菜单切换、功能选择、确认 / 返回）
集成时间显示、传感器交互、小游戏、实用工具等功能，纯硬件 + 嵌入式软件实现，无蓝牙 / WiFi。
功能模块以及具体说明
主页时钟：	RTC 实时时钟显示，按键切换显示样式
菜单系统：滑动动画菜单，7 个功能选项切换
系统设置：时间校准设置
秒表功能：计时、暂停、清零
手电筒：控制 LED 灯亮灭
姿态传感器：MPU6050 读取加速度 / 陀螺仪数据
小恐龙游戏：类似谷歌浏览器的跑酷小游戏
动态表情：OLED 显示自定义表情动画
